# ETPS Logo Assets

This folder contains the official ETPS logo variations for use throughout the application.

## Files

### `etps-logo.png`
**Primary Logo** - Full logo with icon and text
- Use in: Main header, hero section, marketing materials
- Colors: Blue-gray (#475569) primary with teal accent (#0891b2)
- Format: PNG with transparency

### `etps-icon.png`
**Icon Only** - Square icon for favicons and app icons
- Use in: Browser tab (favicon), mobile home screen, small spaces
- Works at: 32x32px to 512x512px
- Format: PNG with transparency

### `etps-wordmark.png`
**Wordmark** - Horizontal layout with tagline
- Use in: Documentation headers, presentations, footer
- Includes: "ETPS" + "Enterprise-Grade Talent Positioning System"
- Format: PNG with transparency

## Usage Guidelines

### Colors
- **Primary:** Blue-gray (#475569, #64748b, #334155)
- **Accent:** Teal (#0891b2, #06b6d4)
- **Background:** Use on white or light backgrounds for best contrast

### Sizing
- **Header:** Use primary logo at ~40-50px height
- **Favicon:** Use icon only at 32x32px or 16x16px
- **Hero:** Use primary logo or wordmark at larger sizes (80-120px height)

### Spacing
- Maintain clear space around logo equal to the height of the "E" in ETPS
- Don't stretch, skew, or alter the logo proportions

## Design System

These logos are part of the ETPS Modern Enterprise design system.

See: `docs/UI_UX_IMPROVEMENT_PLAN.md` for full design specifications.
